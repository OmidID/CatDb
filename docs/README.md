# CatDb Docs

This folder contains the Docusaurus documentation site for CatDb.

## Install

Dependencies are managed with npm and locked in `package-lock.json`.

```bash
npm install
```

## Local development

```bash
npm start
```

## Build

```bash
npm run build
```

The production output is generated in `build/`.

## Type check

```bash
npm run typecheck
```
